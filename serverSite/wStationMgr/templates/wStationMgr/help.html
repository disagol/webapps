API
Nodos

Delete

Descripcion : Borra nodo con id = 8
curl -H "Accept: application/json" -X DELETE http://localhost:8000/wStationMgr/api/8/

Update

Descripcion : Actualiza el titulo para nodo con id = 8
curl -H "Accept: application/json" -X PUT -d "title=Titulo Modificado" http://localhost:8000/wStationMgr/api/8/

Descripcion : Actualiza el titulo para nodo con id = 8 y login
curl -H "Accept: application/json" -X PUT -d "title=Titulo Modificado XXX" -u tester:tester http://localhost:8000/wStationMgr/api/8/

Retrieve

Descripcion : Retorna el nodo con id = 8
curl -H "Accept: application/json" http://localhost:8000/wStationMgr/api/8/

Descripcion : Retorna hijos del nodo sin incluir al mismo nodo con id = 8
curl -H "Accept: application/json" "http://localhost:8000/wStationMgr/api/?name=NASA&includeNode=false"

Create

Descripcion : Crea un servidor asociado al nodo 1
curl -H "Accept: application/json" -X POST -d "title=Server Auto&name=ServerWQ&description=Descripcion del servidor Auto&ip_address=168.120.36.36&os=Windows&parent=1" http://localhost:8000/wStationMgr/api/server/
